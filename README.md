# 📦 Hack the Box Walkthroughs

This repository contains **HackTheBox Walkthroughs**, migrated to **Astro Starlight** for a premium reading experience, modern styling, and automated deployments.

## 🌐 Live Site
The site is automatically built and deployed to GitHub Pages at:
👉 **[https://seclabs-cc.github.io/HackTheBox/](https://seclabs-cc.github.io/HackTheBox/)**

---

## 🚀 Running Locally

To run the documentation site on your local machine:

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the local dev server:**
   ```bash
   npm run dev
   ```
3. Open **[http://localhost:4321/HackTheBox/](http://localhost:4321/HackTheBox/)** in your browser.

For more details, please see the [RUN_LOCALLY.md](RUN_LOCALLY.md) guide.
